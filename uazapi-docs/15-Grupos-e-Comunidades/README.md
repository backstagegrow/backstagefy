# Grupos e Comunidades

Gerenciamento de grupos e comunidades do WhatsApp.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/group/create` | Cria um novo grupo. |
| **POST** | `/group/leave` | Sai de um grupo. |
| **POST** | `/group/delete` | Deleta um grupo. |
| **POST** | `/group/description` | Atualiza a descrição do grupo. |
| **POST** | `/group/subject` | Atualiza o nome/assunto do grupo. |
| **POST** | `/group/picture` | Atualiza a foto do grupo. |
| **POST** | `/group/participant/add` | Adiciona um participante ao grupo. |
| **POST** | `/group/participant/remove` | Remove um participante do grupo. |
| **POST** | `/group/participant/promote` | Promove um participante a administrador. |
| **POST** | `/group/participant/demote` | Rebaixa um administrador a participante. |
| **GET** | `/group/participants` | Lista todos os participantes do grupo. |
| **GET** | `/group/invite-code` | Obtém o código de convite do grupo. |
| **POST** | `/group/invite-code/revoke` | Revoga o código de convite atual. |
| **POST** | `/group/invite/accept` | Aceita um convite para entrar em um grupo. |
| **POST** | `/community/create` | Cria uma nova comunidade. |
| **GET** | `/community/list` | Lista todas as comunidades. |
