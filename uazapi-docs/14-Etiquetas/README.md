# Etiquetas

Gerenciamento de etiquetas/labels.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/label/list` | Lista todas as etiquetas. |
| **POST** | `/label/create` | Cria uma nova etiqueta. |
| **POST** | `/label/delete` | Remove uma etiqueta. |
