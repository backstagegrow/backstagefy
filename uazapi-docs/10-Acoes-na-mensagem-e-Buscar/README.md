# Acoes na mensagem e Buscar

Ações que podem ser realizadas em mensagens existentes.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/message/read` | Marca uma mensagem como lida. |
| **POST** | `/message/delete` | Deleta uma mensagem para todos. |
| **POST** | `/message/edit` | Edita uma mensagem enviada. |
| **POST** | `/message/download-media` | Faz download da mídia de uma mensagem. |
| **POST** | `/message/fetch` | Busca mensagens de um chat. |
| **POST** | `/message/fetch-by-id` | Busca uma mensagem específica pelo ID. |
