# Respostas Rapidas

Gerenciamento de respostas rápidas.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/quick-reply/list` | Lista todas as respostas rápidas cadastradas. |
| **POST** | `/quick-reply/send` | Envia uma resposta rápida. |
