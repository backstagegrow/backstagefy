# Integracao Chatwoot

Integração com a plataforma Chatwoot.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/chatwoot/config` | Configura a integração com Chatwoot. |
| **POST** | `/chatwoot/remove` | Remove a integração com Chatwoot. |
