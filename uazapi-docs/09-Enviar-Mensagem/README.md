# Enviar Mensagem

Endpoints para envio de diferentes tipos de mensagens.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/message/send/text` | Envia uma mensagem de texto. |
| **POST** | `/message/send/image` | Envia uma mensagem com imagem. |
| **POST** | `/message/send/video` | Envia uma mensagem com vídeo. |
| **POST** | `/message/send/audio` | Envia uma mensagem de áudio. |
| **POST** | `/message/send/document` | Envia um documento. |
| **POST** | `/message/send/sticker` | Envia um sticker. |
| **POST** | `/message/send/location` | Envia uma localização. |
| **POST** | `/message/send/contact` | Envia um contato. |
| **POST** | `/message/send/poll` | Envia uma enquete. |
| **POST** | `/message/send/list` | Envia uma mensagem com lista de opções. |
| **POST** | `/message/send/buttons` | Envia uma mensagem com botões. |
