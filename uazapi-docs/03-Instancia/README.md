# Instancia

Operações relacionadas ao ciclo de vida de uma instância, como conectar, desconectar e verificar o status.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/instance/connect` | Inicia a conexão da instância com o WhatsApp. |
| **POST** | `/instance/disconnect` | Desconecta a instância do WhatsApp. |
| **GET** | `/instance/status` | Retorna o status atual da instância. |
| **POST** | `/instance/update-name` | Atualiza o nome da instância. |
| **DELETE** | `/instance/delete` | Remove uma instância permanentemente. |
| **GET** | `/instance/privacy` | Obtém as configurações de privacidade da instância. |
| **POST** | `/instance/privacy` | Atualiza as configurações de privacidade. |
| **POST** | `/instance/presence` | Atualiza o status de presença (online/offline). |
