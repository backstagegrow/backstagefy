# Perfil

Operações relacionadas ao perfil da instância do WhatsApp.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/profile/name` | Atualiza o nome do perfil do WhatsApp. |
| **POST** | `/profile/picture` | Atualiza a foto de perfil do WhatsApp. |
