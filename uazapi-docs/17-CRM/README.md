# CRM

Gerenciamento de campos CRM.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/crm/fields` | Lista todos os campos CRM disponíveis. |
| **POST** | `/crm/update` | Atualiza os campos CRM de um contato. |
