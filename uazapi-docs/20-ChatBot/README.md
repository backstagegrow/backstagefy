# ChatBot

Gerenciamento de fluxos de chatbot.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/chatbot/flow/create` | Cria um novo fluxo de chatbot. |
| **GET** | `/chatbot/flows` | Lista todos os fluxos de chatbot. |
| **GET** | `/chatbot/flow` | Obtém detalhes de um fluxo específico. |
| **POST** | `/chatbot/flow/update` | Atualiza um fluxo existente. |
| **POST** | `/chatbot/flow/delete` | Remove um fluxo de chatbot. |
| **POST** | `/chatbot/activate` | Ativa o chatbot para a instância. |
| **POST** | `/chatbot/deactivate` | Desativa o chatbot. |
| **GET** | `/chatbot/status` | Retorna o status atual do chatbot. |
| **POST** | `/chatbot/webhook` | Endpoint de webhook para o chatbot. |
