# Webhooks e SSE

Configuração de webhooks e Server-Sent Events.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/webhook` | Configura o webhook para a instância. |
| **GET** | `/webhook` | Retorna a configuração atual do webhook. |
| **GET** | `/sse` | Endpoint para conexão SSE em tempo real. |
