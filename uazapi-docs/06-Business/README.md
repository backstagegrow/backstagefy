# Business

⚠️ EXPERIMENTAL - Operações relacionadas ao perfil comercial do WhatsApp Business.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **POST** | `/business/get/profile` | Retorna o perfil comercial da instância. |
| **GET** | `/business/categories` | Lista todas as categorias de negócio disponíveis. |
| **POST** | `/business/update/profile` | Atualiza as informações do perfil comercial. |
| **POST** | `/business/catalog/products` | Lista todos os produtos do catálogo. |
| **POST** | `/business/catalog/product` | Obtém detalhes de um produto específico. |
| **POST** | `/business/catalog/product/delete` | Remove um produto do catálogo. |
| **POST** | `/business/catalog/product/show` | Torna um produto visível no catálogo. |
| **POST** | `/business/catalog/product/hide` | Oculta um produto do catálogo. |
