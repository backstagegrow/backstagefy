# Contatos

Gerenciamento de contatos.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/contact/list` | Lista todos os contatos. |
| **GET** | `/contact` | Obtém informações de um contato. |
| **POST** | `/contact/check-number` | Verifica se um número existe no WhatsApp. |
| **GET** | `/contact/profile-picture` | Obtém a foto de perfil de um contato. |
| **POST** | `/contact/block` | Bloqueia um contato. |
| **POST** | `/contact/unblock` | Desbloqueia um contato. |
