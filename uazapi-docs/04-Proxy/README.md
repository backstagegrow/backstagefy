# Proxy

A uazapiGO opera com um proxy interno como padrão. Você pode configurar um proxy próprio.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/instance/proxy` | Retorna a configuração atual de proxy. |
| **POST** | `/instance/proxy` | Configura um novo proxy para a instância. |
| **DELETE** | `/instance/proxy` | Remove a configuração de proxy. |
