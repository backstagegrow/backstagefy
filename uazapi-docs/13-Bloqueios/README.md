# Bloqueios

Gerenciamento de contatos bloqueados.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/block/list` | Lista todos os contatos bloqueados. |
| **POST** | `/block` | Bloqueia ou desbloqueia um contato. |
