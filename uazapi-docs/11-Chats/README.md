# Chats

Gerenciamento de conversas/chats.

---

## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| **GET** | `/chat/list` | Lista todos os chats. |
| **GET** | `/chat` | Obtém informações de um chat específico. |
| **POST** | `/chat/archive` | Arquiva um chat. |
| **POST** | `/chat/unarchive` | Remove um chat do arquivo. |
| **POST** | `/chat/delete` | Deleta um chat. |
| **POST** | `/chat/clear` | Limpa todas as mensagens de um chat. |
